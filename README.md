# Codez Switch

Codez Switch 是一个面向 Codex / OpenAI 工作流的桌面端管理器。

它把本地反代、账号切换、请求观测、限额状态、配置接管和版本更新整合进一个桌面应用里，目标不是“勉强能用”，而是“长期稳定、可控、可维护”。

## 为什么它更省事

- 本地自带 `cs-proxy` 侧车服务，启动应用就能直接接管请求链路，不用再手动起一堆脚本。
- 账号池和本地使用状态统一管理，切换账号时不需要反复手改 `auth.json`、`current.dat` 或环境变量。
- 请求记录可直接看到状态码、TTFB、流式耗时、输入输出、缓存和错误信息，排查问题不再靠猜。
- 5H / 7D 限额、冷却状态、账号健康度都有可视化反馈，适合多账号长期运行。
- 支持检测并接管本机 Codex 配置，尽量减少用户本地手工配置成本。
- 桌面端支持公告、异常提示、更新检测和安装包升级，适合持续分发给团队成员或客户使用。

## 核心能力

- 本地反代一键启动与状态检测
- 多账号库存、切换、禁用、删除联动
- 请求日志与用量统计
- 账号限额探测与冷却感知
- Codex 本地配置自动接管
- 管理密钥绑定与本机恢复
- 公告弹窗与强制通知
- 桌面端版本检测与自动下载安装
- Windows / macOS / Linux 多端分发

## 适合什么场景

- 需要长期稳定使用 Codex，而不是临时跑一下命令行
- 需要管理多账号、多用户或多台机器
- 需要知道“为什么这次请求慢 / 失败 / 冷却 / 被限流”
- 需要把客户端和本地代理打包成统一安装包对外分发

## 下载

请直接前往本仓库的 Releases 页面下载对应平台安装包。

- Windows: `.msi`

- macOS Apple Silicon: `.dmg`

## 自动更新

应用支持版本检测与更新提示。

- 启动后可静默检查新版本
- 检测到更新后可直接下载对应平台安装包
- 下载完成后自动拉起安装器
- 更新源由服务端统一配置，便于集中分发
